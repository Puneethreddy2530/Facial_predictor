üìä DATASET INFORMATION:
============================================================

UTKFace:
  ‚Ä¢ 23,708 face images
  ‚Ä¢ Labels: age (0-116), gender (M/F), race (5 classes)
  ‚Ä¢ File format: [age]_[gender]_[race]_[timestamp].jpg
  ‚Ä¢ Source: https://susanqq.github.io/UTKFace/

FER2013:
  ‚Ä¢ 35,887 grayscale faces
  ‚Ä¢ Labels: 7 emotions (angry, disgust, fear, happy, sad, surprise, neutral)
  ‚Ä¢ Requires Kaggle API
  ‚Ä¢ Source: Kaggle FER2013

LFW (Sample):
  ‚Ä¢ 13,000+ labeled faces
  ‚Ä¢ Good for quick testing
  ‚Ä¢ Source: http://vis-www.cs.umass.edu/lfw/

üéØ MODEL ARCHITECTURE:
============================================================

Age Model (Regression):
  ‚Ä¢ Input: 128x128x3 RGB image
  ‚Ä¢ 4 Conv blocks with BatchNorm + Dropout
  ‚Ä¢ Output: Single value (age in years)
  ‚Ä¢ Loss: MSE, Metric: MAE

Gender Model (Binary Classification):
  ‚Ä¢ Input: 128x128x3 RGB image
  ‚Ä¢ 3 Conv blocks + Dropout
  ‚Ä¢ Output: Binary (0=male, 1=female)
  ‚Ä¢ Loss: Binary crossentropy

Race Model (Multi-class):
  ‚Ä¢ Input: 128x128x3 RGB image
  ‚Ä¢ 3 Conv blocks + Dropout
  ‚Ä¢ Output: 5 classes (White, Black, Asian, Indian, Others)
  ‚Ä¢ Loss: Sparse categorical crossentropy
